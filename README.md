# Process-tools

Инструменты для поддержки и автоматизации личного рабочего процесса.

Репозиторий содержит скрипты, шаблоны и утилиты, которые помогают
собирать данные, формировать заметки и упрощать работу с другими репозиториями.

## Назначение
- автоматизация рутинных действий
- генерация markdown-файлов (daily / weekly / monthly)
- интеграции с GitHub, Obsidian и Telegram

## Структура
- `scripts/` — исполняемые скрипты и боты
- `templates/` — шаблоны markdown
- `configs/` — конфигурации (без секретов)
- `logs/` — технические логи выполнения
- `rules.md` — правила и ограничения

## Принципы
- инструменты не хранят данные прогресса
- секреты не хранятся в репозитории
- автоматика обслуживает процесс
